DATA Y VISUALIZACIÓN
- Para poder acceder al archivo "Proyecto Diplomado", se necesita previamente descargar e instalar Power BI Desktop
- El link de la página de descarga es el siguiente:
https://www.microsoft.com/es-es/power-platform/products/power-bi/desktop
- Una vez concluida la instalación, y abierto el programa, seleccionar la opción "Open"
- Elegir el archivo "Proyecto Diplomado.pbix"
DESARROLLO
- El archivo "Proyecto Winsorización" puede ser abierto usando la aplicación Collab Notebook de Google.
- Para acceder a esta herramienta se debe ingresar al siguiente link:
https://www.microsoft.com/es-es/power-platform/products/power-bi/desktop
- Una vez haya cargado la página existe la opción de subir o cargar un archivo, elegir el archivo "Proyecto Winsorización.ipynb"
- Luego de acceder al archivo, elegir la opción "Entorno de ejecución", y luego "Ejecutar todo", esto permitira que todo el código realizado sea ejecutado, y se podran observar los resultados.
